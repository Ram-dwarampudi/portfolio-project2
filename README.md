## Setup Instructions

### 1. Download Files
Save all files to a folder on your computer:
```
portfolio-folder/
├── index.html
├── styles.css
├── script.js
├── profile.jpg (you need to add this)
└── README.md
```

### 2. Add Your Profile Photo
- Take or find a professional photo of yourself
- Name it "profile.jpg" 
- Place it in the same folder as the other files
- Recommended size: 400x400 pixels, square format

### 3. Open in Browser
- Double-click on `index.html` to open in your web browser
- Or right-click and choose "Open with" → your preferred browser

## Deployment Options

### Option 1: Free Hosting Services
- **Netlify**: Drag and drop your folder to netlify.com
- **Vercel**: Connect to GitHub and deploy automatically
- **GitHub Pages**: Upload to GitHub repository and enable Pages

### Option 2: Web Hosting
- Upload files to any web hosting service via FTP
- Ensure all files are in the root directory or public_html folder

### Option 3: Local Development
- Use VS Code with Live Server extension for local testing
- Python: `python -m http.server 8000`
- Node.js: `npx serve .`
